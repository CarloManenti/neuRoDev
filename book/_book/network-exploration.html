2 Network exploration

2.1 Loading necessary libraries and data

This chapter demonstrates how to explore the reference networks of corticogenesis, neurogenesis and gliogenesis. It provides the code chunks to reproduce the results of the paper figures and examples on how to use the resource. The data used can be downloaded here as an r package, in which we put all the data we leveraged in our analyses. If someone wants to start from the count matrices, we have stored them as a unique matrix here … and the associated metadata here …

2.1.2 Loading the networks necessary objects

We extract the data stored in neuRoDevDB. For each reference network we have:

  1. df: here are saved all the information about the networks, including the name of each cluster, the study, its age in days and stages, its subclass/subtype, starting class and subclass annotation (RefClass and RefSubClass), x and y UMAP coordinates, etc.
  2. umap: it comprises the network (umap_out), both knn and snn graphs, the number of neighbors and components used, etc.
  3. palette: to visualize the network with the same color code as we show in our figures, we included the network-specific palette and the stage-specific one.
Show the code (lineage-specific objects) 
#corticogenesis-specific objects
corticogenesis_df <- corticogenesis_df()
corticogenesis_umap <- corticogenesis_umap()
corticogenesis_palette <- corticogenesis_palette()

#neurogenesis-specific objects
neurogenesis_df <- neurogenesis_df()
neurogenesis_umap <- neurogenesis_umap()
neurogenesis_palette <- neurogenesis_palette()

#gliogenesis-specific objects
gliogenesis_df <- gliogenesis_df()
gliogenesis_umap <- gliogenesis_umap()
gliogenesis_palette <- gliogenesis_palette()

#stage-specific palette
stages_palette <- stages_palette()

With this we can start the analysis and the exploration of the reference networks.

2.2 Network visualizations

2.2.1 SubClass (or SubType) labels and colors

Here we show how it is possible to visualize in the network the different cell types and the age of each cluster. First, we need to obtain the UMAP plot of the network, which is build upon the correlations to the reference signatures. To color the clusters we can provide the labels and their associated color, thus making possible to visualize any kind of label.

The network-specific dataframes retain information such as the cluster name, the stage, the age in days, the original label, our annotations etc. Here we show just part of this table:

Subclass-based colors in corticogenesis, neurogenesis and gliogenesis networks. umap_signature_plot recreates the plots in figure 2, figure 5, and figure 6. The inputs are:

  • signatures_cor = the dataframe with all the information, including the correlation to the reference signatures
  • color_attr = the column used to define the groups to plot
  • col_vector = the named vector of colors based on color_attr
  • umap_obj = the already computed umap for the networks

The output will comprise:

  • umap_obj = it is a list that includes all the data necessary to build the network
  • umap_plot = the actual network plot
corticogenesis_subclass_plot <- umap_signature_plot(signatures_cor = corticogenesis_df, 
                                    color_attr = 'SubClass', 
                                    col_vector = corticogenesis_palette, 
                                    umap_obj = corticogenesis_umap)

neurogenesis_subtype_plot <- umap_signature_plot(signatures_cor = neurogenesis_df,
                                    color_attr = 'SubType', 
                                    col_vector = neurogenesis_palette, 
                                    umap_obj = neurogenesis_umap)

gliogenesis_subtype_plot <- umap_signature_plot(signatures_cor = gliogenesis_df, 
                                    color_attr = 'SubType', 
                                    col_vector = gliogenesis_palette, 
                                    umap_obj = gliogenesis_umap)

To actually plot the network, we can just run the umap_plot object:

2.3 SubClass and SubType composition across ages

Cortical development is an highly dynamic process, with cell types emerging at different stages of life due to progenitors changing their fate based on the developmental period. To explore how this happens, we can look at the cell type composition in time.

2.3.1 Proportions across ages

To assess how the subclasses and subtypes change in time, we can take a look at how their proportions evolve, leveraging our knowledge of both cell type and time after conception.

Show the code (ordering age) 
unique_ages <- mixedsort(unique(corticogenesis_df$Age))
ordered_ages <- c(unique_ages[grep('w', unique_ages)], unique_ages[grep('d', unique_ages)], unique_ages[grep('m', unique_ages)], unique_ages[grep('yr', unique_ages)])

stages.indexes <- unlist(lapply(names(stages_palette), function(i) {
  strsplit(i, "-")[[1]][1]
  }))
Show the code (lineages proportion in time) 
corticogenesis_subclass_proportion <- table(corticogenesis_df$SubClass, corticogenesis_df$Age)

corticogenesis_subclass_proportion <- t(t(corticogenesis_subclass_proportion)/colSums(corticogenesis_subclass_proportion))


neurogenesis_subtype_proportion <- table(neurogenesis_df$SubType, neurogenesis_df$Age)

neurogenesis_subtype_proportion <- t(t(neurogenesis_subtype_proportion)/colSums(neurogenesis_subtype_proportion))


gliogenesis_subtype_proportion <- table(gliogenesis_df$SubType, gliogenesis_df$Age)

gliogenesis_subtype_proportion <- t(t(gliogenesis_subtype_proportion)/colSums(gliogenesis_subtype_proportion))

Taking as an example corticogenesis, here we show the dynamics of cell type development. Radial glia and early immature neurons) are high in the first time points, and decrease later on, while mature neurons show the opposite trend. Notably, deep layer neurons appear first, while upper layers emerge later. Moreover, glia subtypes are present already at early stages (OPCs and astrocytes), and that only later on are present oligodendrocytes too.

barplot(corticogenesis_subclass_proportion[,ordered_ages], 
        col = corticogenesis_palette[rownames(corticogenesis_subclass_proportion)], 
        las = 2, 
        ylab = 'SubClass proportion')

Figure 2.7: Corticogenesis Subtypes proportions

Increasing the resolution and going to the neuron-specific network, we can observe that progenitors start to decrease around the peak of neurogenesis (w14-16), while mature neurons are predominant from birth on. Furthermore, the neuronal subtypes we define depict the inside-out stereotypical way of cortical development. Deep non intratelencephalic (Non-IT) are followed by intratelencephalic (IT) ones, with the upper layer IT neurons (ExN-L2-3-IT) being the last to be produced.

barplot(neurogenesis_subtype_proportion[,ordered_ages], 
        col = neurogenesis_palette[rownames(neurogenesis_subtype_proportion)], 
        las = 2, 
        ylab = 'SubType proportion')

Figure 2.8: Neurogenesis Subtypes proportions

The gliogenesis network, instead, allowed us to describe more fine radial glia subtypes. In early stages we can observe a variety of progenitors, with a nice distinction between different cell cycle phases (G2M and S). This process comprise two major lineages: astrogenesis and oligodendrogenesis. Cell types of these two branches appear already between w10 and w18. Immature OPCs coming from the ganglionic eminences migrate towards the cortex, followed by radial glia generating astrocytes after concluding neurogenesis (peak w18-w25).

barplot(gliogenesis_subtype_proportion[,ordered_ages[which(ordered_ages %in% colnames(gliogenesis_subtype_proportion))]],
        col = gliogenesis_palette[rownames(gliogenesis_subtype_proportion)], 
        las = 2, 
        ylab = 'SubType proportion')

2.3.2 Compute the cumulative cluster number proportion

Another way of looking cell type dynamics during cortical development involves examining the number of clusters assigned to each subtype across developmental stages

Show the code (subtype cumulative sum by lineage) 
# here for each row (i.e. for each subclass/subtype) 
# ordered by age, we divide the cumulative sum of 
# clusters by the total number of clusters to obtain 
# subclass/subtype proportion for each stage

corticogenesis_subtype_cumsum <- t(
  apply(X = table(corticogenesis_df$SubClass, corticogenesis_df$Stages)[,mixedsort(unique(corticogenesis_df$Stages))],  
        MARGIN =  1, 
        FUN = function(i) {
  cumsum(i/sum(i)) 
})) 

neurogenesis_subtype_cumsum <- t(apply(table(neurogenesis_df$SubType, neurogenesis_df$Stages)[,mixedsort(unique(neurogenesis_df$Stages))], 1, function(i) {
  cumsum(i/sum(i))
}))

gliogenesis_subtype_cumsum <- t(apply(table(gliogenesis_df$SubType, gliogenesis_df$Stages)[,mixedsort(unique(gliogenesis_df$Stages))], 1, function(i) {
  cumsum(i/sum(i))
}))

The cumulative sum gives us different levels of information. First, we can observe the time point in which a cell type starts to populate the cortex; second, we can see when the proportion of clusters increases the fastest, showing a prevalence of generation or maturation of the subtypes; third, we can appreciate the period of time in which each cell type reaches is plateau, i.e. is not generated anymore or has fully matured. For example: radial glia generally reach their plateau before birth, and in a similar way behave InN-Immature, which in time leaves the place to mature inhibitory neurons. Oligodendrocytes, instead, appear postnatally with a peak around late childhood and only reach their plateau in adulthood, in accordance with the processes of synaptogenesis and myelination.

Show the code (plotting corticogenesis) 
par(mar=c(4.2,3,1,1))
plot(corticogenesis_subtype_cumsum[1,], 
     type = 'l',
     las = 1,
     lwd = 2, 
     col = corticogenesis_palette[rownames(corticogenesis_subtype_cumsum)[1]], 
     xlab = 'Stage index', 
     ylab = 'Cumulative cluster proportion', 
     ylim = c(min(corticogenesis_subtype_cumsum), max(corticogenesis_subtype_cumsum)),
     xaxt="n")
axis(side = 1, 
     labels = stages.indexes,
     at = 1:length(stages.indexes),
     las = 1)
for(i in seq(2, nrow(corticogenesis_subtype_cumsum))) {
  lines(corticogenesis_subtype_cumsum[i,], 
        las = 1,
        lwd = 2.5, 
        col = corticogenesis_palette[rownames(corticogenesis_subtype_cumsum)[i]])
}

2.4 Visualize the expression of genes or gene sets

For each cluster we computed its pseudobulk profile. With this, we can explore gene expression in the network.

Show the code (lineage-specific pseudobulk) 
corticogenesis_pseudobulk <- corticogenesis_pseudobulk()
neurogenesis_pseudobulk <- neurogenesis_pseudobulk()
gliogenesis_pseudobulk <- gliogenesis_pseudobulk()

We can plot gene expression either raw or after smoothing, which makes gene expression more homogeneous between clusters in the same neighborhood.

# no smoothing - one gene - no labels
plotGenesRank(umap_obj = corticogenesis_umap, 
              genes = 'VIP', 
              refExpression = logcounts(corticogenesis_pseudobulk), 
              smooth = FALSE,
              title = "raw expression")

#smoothing considering 20 neighbors - one gene
plotGenesRank(umap_obj = corticogenesis_umap, 
              genes = 'VIP', 
              refExpression = logcounts(corticogenesis_pseudobulk), 
              smooth = TRUE,
              n_nearest_smooth = 20, 
              label_attr = corticogenesis_df$SubClass,
              title = "smoothed expression")
